📚 Bookstore Analytics with SQL
🚀 Project Overview

This project demonstrates a complete end-to-end SQL analytics workflow using a relational database for a bookstore business.

The goal is to analyze:

📈 Book performance

🛒 Customer purchasing behavior

💰 Marketing ROI

📦 Inventory alerts

🔁 Customer retention & churn

📊 Monthly revenue trends

This project simulates real-world tasks performed by a Data Analyst in retail/e-commerce companies.

🛠 Tools & Technologies

SQL (SQLite / MySQL compatible)

Joins

CTEs (Common Table Expressions)

Subqueries

Aggregations

Date functions

RFM Analysis

Business KPI calculations

🗄 Database Schema

The project contains four relational tables:

Books

Customers

Orders

MarketingSpend

Entity Relationships

One customer → Many orders

One book → Many orders

One customer → One marketing spend record
